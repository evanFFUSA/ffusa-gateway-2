Professional FFUSA payment gateway integration for WordPress and WooCommerce.

== Description ==

Complete FFUSA (NMI) payment gateway integration for WordPress websites provided by FFUSA. Supports both standalone payment forms and WooCommerce integration.

**Features:**
* Direct payment processing
* Secure payment handling
* Refund support
* Sandbox mode for testing
* Easy admin configuration

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings â†’ NMI Gateway to configure
4. Enter your NMI credentials
5. Enable the gateway
6. Be sure that all fields present are also the only ones checked on the processor

== Frequently Asked Questions ==

= Do I need an NMI account? =
Yes, you need an active NMI merchant account with API credentials.

= Is this secure? =
Yes, all payment data is transmitted securely to NMI servers.
